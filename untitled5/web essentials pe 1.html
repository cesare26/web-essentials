Wie ben ik

ik ben Cesare Calianno wooon in Hasselt ben 23 jaar heb 1 broer van 32 jaar ik wil al langer als vandaag een eigen zaak starten een goede renderende zaak waar ik goed mijn brood kan mee verdienen.In deze tijd is het zo moeilijk om dat te waar te maken. Had nooit gedacht dat Progreren zwaar zouw zijn probeer wel mijn best te doen meer kan ik niet doen op dit moment heb ik soort black-out kan niks nietmeer onthouden.Ben uit men ritme weet ook niet dat ooit terug komt. Mijn oude ritme waar ik snelheid en drukte kan combineren. en voor het programeren deet ik toegepaste informatica was ook zwaar heb voor Programeren gekozen om dat ik daar werk voor is ben als Grafisch Ontwerper afgestudeerd. maar helaas vind ik er geen werk in en het spijtige is dat ik heel goed kan werken met mijn handen dat ik niet slim genoeg ben om programeren aan te kunnen heb nooit zo een code taal gezien in mijn heel leven zelf mijn ouders verschieten ervan wat ik aan kan maar.Weet zelf niet of ik dit jaar wel haal vanweg de druk en kan zoveel input onmogelijk allemaal onthouden.
